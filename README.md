# PyDanny-V2

PyDanny.com V2 blog

## Getting Started

to get started:

```zsh
git clone https://github.com/pydanny/pydanny-v2.git

cd pydanny-v2

yarn

yarn run dev
```

### Prerequisites

its good to have Vuepress installed globally on your machine


```
yarn add -D vuepress
```
